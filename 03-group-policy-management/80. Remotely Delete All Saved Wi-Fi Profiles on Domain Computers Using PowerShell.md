# 📡 Remotely Delete All Saved Wi-Fi Profiles on Domain Computers Using PowerShell

## 📘 Overview

This guide explains how to remotely remove all saved Wi-Fi profiles from domain-joined computers using PowerShell and Active Directory.

The process uses PowerShell remoting to execute `netsh wlan delete profile` commands on all computers retrieved from Active Directory.

---

# 🖥️ Prerequisites

Before starting, ensure the following requirements are met:

* ✅ The target computers are **domain joined**
* ✅ PowerShell remoting is enabled on client computers
* ✅ Active Directory module is installed on the server
* ✅ You have **Domain Administrator** or equivalent administrative privileges
* ✅ Target devices are online and reachable

---

# 🔍 Verify Existing Saved Wi-Fi Profiles on a Domain Computer

## Step 1 — Open Wi-Fi Settings

Go to the domain-joined computer.

Navigate to:

```text
Settings > Network & Internet > Wi-Fi > Manage Known Networks
```

## Step 2 — Review Saved Wi-Fi Profiles

Here, you can see all the saved Wi-Fi profiles currently stored on the computer.

---

# 🛠️ Run the PowerShell Script from the Active Directory Server

## Step 1 — Access the Active Directory Server

Go to the Active Directory server.

## Step 2 — Open PowerShell as Administrator

Open:

```text
PowerShell > Run as Administrator
```

---

# 📜 Original PowerShell Command

> ⚠️ The following script retrieves all domain computers from Active Directory and deletes all saved Wi-Fi profiles remotely.

```powershell
$Computers = Get-ADComputer -Filter * | Select-Object -ExpandProperty Name

foreach ($PC in $Computers) {
    Invoke-Command -ComputerName $PC -ScriptBlock {
        $profiles = netsh wlan show profiles | Select-String "All User Profile" | ForEach-Object {
            ($_ -split ":")[1].Trim()
        }

        foreach ($profile in $profiles) {
            netsh wlan delete profile name="$profile"
        }
    } -ErrorAction SilentlyContinue
}
```

---

# ▶️ Execute the Script

## Step 1 — Run the Command

Paste the script into PowerShell.

Press:

```text
Enter
```

## Step 2 — Wait for Completion

Wait until the script finishes executing.

The script will:

* Retrieve all domain computers
* Connect remotely to each computer
* Enumerate saved Wi-Fi profiles
* Delete all detected Wi-Fi profiles

---

# ✅ Verify Successful Wi-Fi Profile Deletion

After the code runs successfully on the domain-joined computers:

* The PowerShell window will display deleted Wi-Fi profile information.
* All saved Wi-Fi profiles should be removed from the target devices.

---

# 🔎 Confirm Removal on the Domain Computer

Go back to the domain-joined computer.

Navigate again to:

```text
Settings > Network & Internet > Wi-Fi > Manage Known Networks
```

You should now see the following message:

```text
No known networks yet. Connect to a network, then check back here.
```

---

# 📌 Notes

> 💡 This operation permanently removes all saved Wi-Fi profiles from the target computers.

> ⚠️ Users will need to reconnect manually to Wi-Fi networks after profile deletion.

> 🔒 Consider testing the script on a small group of computers before deploying domain-wide.

---

# 🧪 Recommended Validation

You can manually validate profile removal on a target machine using:

```powershell
netsh wlan show profiles
```

If successful, no saved Wi-Fi profiles should appear.
