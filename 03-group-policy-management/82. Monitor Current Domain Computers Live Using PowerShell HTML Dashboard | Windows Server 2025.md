# How to Monitor Current Domain Computers Live Using PowerShell HTML Dashboard | Windows Server 2025

## 📌 Overview

This guide explains how to create a **live monitoring dashboard** for domain-joined computers using **PowerShell** and an **HTML dashboard** on **Windows Server 2025**.

The dashboard will display:

* 💻 Computer Name
* 🟢 Online / 🔴 Offline Status
* 👤 Logged-in User
* 🕒 Last Update Time

The dashboard automatically refreshes every 10 seconds and updates the status of domain computers in real time.

---

# 🛠️ Requirements

Before starting, ensure the following requirements are met:

* Windows Server 2025
* Active Directory Domain Services (AD DS)
* PowerShell running as Administrator
* Domain-joined computers
* Network connectivity between systems
* Active Directory PowerShell Module installed

---

# 📂 Step 1 — Open the AD Server

Open the **Active Directory (AD) Server**.

---

# 💾 Step 2 — Create Dashboard Folder

Let us suppose we have 2 drives:

* `Local Disk (C:)`
* `New Volume (D:)`

Open the `New Volume (D:)` drive.

Create a new folder:

```text
D:\Dashboard
```

---

# ⚡ Step 3 — Open PowerShell as Administrator

1. Open **PowerShell**
2. Right-click it
3. Select:

```text
Run as Administrator
```

---

# 📋 Step 4 — Paste the PowerShell Script

Paste the following command into PowerShell.

---

# 🧾 Original Command

```powershell
# Get all domain computers
$computers = Get-ADComputer -Filter * | Select-Object -ExpandProperty Name

# Output file
$path = "D:\Dashboard\DomainDashboard.html"

# Create folder if not exists
if (!(Test-Path "C:\Dashboard")) {
    New-Item -Path "C:\Dashboard" -ItemType Directory
}

while ($true) {

    $results = foreach ($pc in $computers) {

        $online = Test-Connection -ComputerName $pc -Count 1 -Quiet -ErrorAction SilentlyContinue

        if ($online) {
            try {
                $cs = Get-CimInstance Win32_ComputerSystem -ComputerName $pc

                [PSCustomObject]@{
                    ComputerName = $pc
                    Status       = "<span class='online'>Online</span>"
                    UserName     = $cs.UserName
                    LastUpdate   = (Get-Date -Format "yyyy-MM-dd HH:mm:ss")
                }
            }
            catch {
                [PSCustomObject]@{
                    ComputerName = $pc
                    Status       = "<span class='error'>Error</span>"
                    UserName     = "-"
                    LastUpdate   = (Get-Date -Format "yyyy-MM-dd HH:mm:ss")
                }
            }
        }
        else {
            [PSCustomObject]@{
                ComputerName = $pc
                Status       = "<span class='offline'>Offline</span>"
                UserName     = "-"
                LastUpdate   = (Get-Date -Format "yyyy-MM-dd HH:mm:ss")
            }
        }
    }

# HTML Design
$html = @"
<html>
<head>
<title>Domain Dashboard</title>

<meta http-equiv='refresh' content='10'>

<style>
body {
    font-family: Arial;
    background-color: #0f172a;
    color: white;
    text-align: center;
}

h1 {
    margin-top: 20px;
}

table {
    border-collapse: collapse;
    width: 95%;
    margin: auto;
    margin-top: 20px;
    background-color: #1e293b;
}

th {
    background-color: #334155;
    padding: 10px;
}

td {
    padding: 8px;
    border-bottom: 1px solid #475569;
}

tr:hover {
    background-color: #475569;
}

.online {
    color: #22c55e;
    font-weight: bold;
}

.offline {
    color: #ef4444;
    font-weight: bold;
}

.error {
    color: orange;
    font-weight: bold;
}
</style>

</head>
<body>

<h1>💻 Domain Computers Live Dashboard</h1>
<p>Last Refresh: $(Get-Date -Format "yyyy-MM-dd HH:mm:ss")</p>

<table>
<tr>
<th>Computer Name</th>
<th>Status</th>
<th>Logged User</th>
<th>Last Update</th>
</tr>
"@

# Add rows
foreach ($row in $results) {
    $html += "<tr>
<td>$($row.ComputerName)</td>
<td>$($row.Status)</td>
<td>$($row.UserName)</td>
<td>$($row.LastUpdate)</td>
</tr>"
}

$html += @"
</table>

</body>
</html>
"@

# Save file
$html | Out-File $path -Encoding UTF8

Start-Sleep -Seconds 10
}
```

---

# ▶️ Step 5 — Execute the Script

After pasting the script:

1. Click **Enter**
2. Wait until the process finishes

The script will continuously monitor all domain computers and generate the HTML dashboard automatically.

After a moment, the system will create the following file:

```text
D:\Dashboard\DomainDashboard.html
```

The file is created in `.html` format.

# 🌐 Step 7 — Open the Dashboard in a Web Browser

Open the generated HTML file using any web browser.

Example:

# 🧪 Step 8 — Test Offline Computer Status

Test the other joined computer by turning off its network status.

Then come back to the dashboard.

The computer name and its status will be updated automatically.

Example:

```text
Online  →  Offline
```

# 🧪 Step 9 — Test Online Computer Status

Test the other joined computer by turning on its network status.

Then come back to the dashboard.

The computer name and its status will be updated automatically again.

Example:

```text
Offline  →  Online
```
