# 🎧 Remote Media File Detection with PowerShell

This guide explains how the **IT Department** can remotely search **audio and video media files** across all domain computers using **PowerShell** and generate a **report exported to Excel (CSV format)**.

The script queries **Active Directory computers**, remotely scans local drives, and produces a centralized report containing:

* 💻 Computer Name
* 📁 File Path

This is useful for:

* IT compliance auditing
* Storage monitoring
* Identifying unauthorized media files
* Security investigations

---

# 📋 Requirements

Before running the script, ensure the following requirements are met:

* 🖥️ Domain environment with **Active Directory**
* 🔑 Administrative privileges
* ⚡ PowerShell remoting enabled
* 🗂️ Access to **Server Manager**
* 📊 Microsoft Excel or compatible software for viewing CSV reports

---

# 🛠 Procedure

## 1️⃣ Open Server Manager

Open **Server Manager** on your domain server.

---

## 2️⃣ Launch PowerShell as Administrator

Open PowerShell with administrative privileges.

```
open powershell > run as an administrator
```

This is required to execute remote commands across domain computers.

---

# 🔎 Step 1 — Retrieve All Domain Computers

Paste the first command line:

```powershell
$computers = Get-ADComputer -Filter * | Select-Object -ExpandProperty Name
```

### 📘 Description

This command retrieves **all computer names from Active Directory** and stores them in the `$computers` variable.

Each computer will be scanned remotely in the next step.

---

# 🎬 Step 2 — Search Media Files on All Computers

Paste the second command line:

```powershell
$fileTypes = @("*.mp4")
$results = @()

foreach ($computer in $computers) {
    try {
        Invoke-Command -ComputerName $computer -ScriptBlock {
            param($fileTypes)

            $drives = Get-WMIObject -Class Win32_LogicalDisk | Where-Object { $_.DriveType -eq 3 }
            $computerResults = @()

            foreach ($drive in $drives) {
                $driveLetter = $drive.DeviceID
                Write-Host "Searching in $driveLetter on $env:COMPUTERNAME..."

                
                $matchingFiles = Get-ChildItem -Path $driveLetter -Recurse -Include $fileTypes -ErrorAction SilentlyContinue
                
               
                if ($matchingFiles) {
                    $matchingFiles | ForEach-Object {
                        $computerResults += [PSCustomObject]@{
                            ComputerName = $env:COMPUTERNAME
                            FilePath = $_.FullName
                        }
                        Write-Host "Found: $($_.FullName) on $env:COMPUTERNAME"
                    }
                } else {
                    Write-Host "No audio/video files found on $driveLetter of $env:COMPUTERNAME."
                }
            }

            return $computerResults
        } -ArgumentList $fileTypes | ForEach-Object {
            $results += $_
        }
    } catch {
        Write-Host ("Error occurred on computer: " + $computer + " - " + $_.Exception.Message)
    }
}
```

---

## 📘 Script Explanation

This script performs the following operations:

### 💻 Computer Enumeration

Loops through all domain computers stored in `$computers`.

### 🧠 Remote Execution

Uses:

```
Invoke-Command
```

to execute commands remotely on each computer.

### 💽 Drive Detection

Identifies all **local hard drives** using:

```
Win32_LogicalDisk
```

### 🔍 File Scanning

Recursively searches each drive for:

```
*.mp4
```

You can modify the script to include additional file types such as:

```
*.mp3
*.avi
*.mkv
*.wav
```

### 📦 Result Collection

For each matching file found, the script records:

* Computer Name
* Full File Path

---

# 📊 Step 3 — Export Results to CSV (Excel Report)

Paste the third command line:

```powershell
$results | Select-Object ComputerName, FilePath | Export-Csv -Path "C:\Path\To\Your\Output.csv" -NoTypeInformation
```

### 📘 Description

This command exports the collected data into a **CSV file**, which can be opened using **Microsoft Excel**.

The CSV file will contain two columns:

* ComputerName
* FilePath
