# Contributing

Thank you for contributing!

## How to Contribute

1. Fork the repository
2. Create a branch
3. Add your changes
4. Submit a Pull Request

## Documentation Style

Use:

- Markdown
- Clear headings
- Step-by-step instructions
- Screenshots when possible

## Naming Convention

Use lowercase with hyphens.

Example:

user-management.md
group-policy-basics.md
