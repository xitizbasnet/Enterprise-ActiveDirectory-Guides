# 🚫 OneDrive Application Restriction Guide

## Blocking OneDrive for Domain Users Using Group Policy

## 📘 Document Overview

This document provides a structured and professional procedure for blocking the Microsoft OneDrive application for domain users using Group Policy in a Windows Server 2025 Active Directory environment.

---

## 🎯 Objective

To centrally restrict the usage of the OneDrive application for file storage across domain-joined client computers using Group Policy.

---

## 🏗️ Environment Details

| Component       | Value                                            |
| --------------- | ------------------------------------------------ |
| Domain Name     | `xitiztechservices.local`                        |
| Server OS       | Windows Server 2025                              |
| Management Tool | Group Policy Management Console (GPMC)           |
| Target Scope    | Organizational Unit (OU) containing domain users |
| Policy Type     | Computer Configuration Policy                    |

---

# 🛠️ Implementation Procedure

---

## 🔹 Step 1: Open Group Policy Management

1. Open **Server Manager**
2. Click **Tools**
3. Select **Group Policy Management**

✅ The Group Policy Management Console (GPMC) will open.

---

## 🔹 Step 2: Create a New Group Policy Object (GPO)

📍 Navigate to the Organizational Unit (OU) where the target users are located.

Example Domain:

```
xitiztechservices.local
```

### Procedure:

1. Expand the domain.
2. Go to **Group Policy Objects**
3. Right-click → **New**
4. Enter the GPO Name:

```
Deny One drive access
```

5. Right-click the newly created GPO
6. Select **Edit**

✅ The Group Policy Management Editor will open.

---

## 🔹 Step 3: Configure OneDrive Restriction Policy

### Navigate to:

```
Computer Configuration
   └── Policies
       └── Administrative Templates
           └── Windows Components
               └── OneDrive
```

### Configure the following setting:

**Policy Name:**

```
Prevent the usage of OneDrive for file Storage
```

### Configuration Steps:

1. Double-click the policy
2. Select **Enable**
3. Click **Apply**
4. Click **OK**

🔒 This setting prevents OneDrive from being used for file storage across the system.

---

## 🔹 Step 4: Apply Group Policy Update on Server

Open **PowerShell** → Run as Administrator

Execute:

```powershell
gpupdate /force
```

✅ This forces the server to update the applied Group Policies.

---

## 🔹 Step 5: Apply Group Policy Update on Client Computer

Go to the client computer (Domain-joined machine).

Open **PowerShell** → Run as Administrator

Execute:

```powershell
gpupdate /force
```

✅ This ensures the client receives and applies the updated policy immediately.

---

# 🔎 Verification

After applying the policy:

* The client computer may still display a folder named **"OneDrive"**
* However:

  * Opening the OneDrive folder will not function
  * Launching the OneDrive application will not open

✅ This confirms that OneDrive access has been successfully blocked via Group Policy.

---

# 🧾 Expected Behavior

| Component            | Result     |
| -------------------- | ---------- |
| OneDrive Folder      | Visible    |
| OneDrive Application | Blocked    |
| File Sync            | Disabled   |
| User Access          | Restricted |

# 🔐 Security Impact

* Prevents unauthorized cloud file storage usage
* Enforces centralized IT storage policies
* Improves compliance and data governance
* Reduces risk of external file synchronization

---

# 📌 Best Practices

* Always link the GPO to the correct OU
* Test in a staging OU before production deployment
* Document GPO changes in change management records
* Use consistent naming conventions for GPOs

---

# 🏁 Conclusion

The OneDrive application is now restricted for domain users using Group Policy in Windows Server 2025. This configuration ensures centralized administrative control and prevents unauthorized file storage via OneDrive within the organization.
