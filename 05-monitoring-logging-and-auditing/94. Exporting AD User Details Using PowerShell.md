# 📖 Introduction

This guide provides a professional step-by-step process for exporting Microsoft Active Directory (AD) user information using Windows PowerShell.

Using PowerShell allows administrators to efficiently:

* Export all Active Directory users
* Generate CSV-based reports
* Retrieve custom user attributes
* Filter users by Organizational Units (OU)
* Export active or disabled accounts
* Audit and document Active Directory environments

The exported data can be used for:

* User inventory management
* Security audits
* Compliance reporting
* HR synchronization
* Access reviews
* Backup documentation

# 🧩 Chapter 2 — Exporting Active Directory Users

# Step 3 — Export All Active Directory Users

Use the following PowerShell command to retrieve all Active Directory users and export them into a CSV file.

---

## Standard Export Command

```powershell id="m7u4ra"
Get-ADUser -Filter * -Property * |
Select-Object SamAccountName, Name, GivenName, Surname, EmailAddress, Department, Title, Enabled |
Export-Csv -Path "C:\AD_Users.csv" -NoTypeInformation
```

---

# 📘 Command Explanation

---

## Retrieve All Users

```powershell id="j8q2yn"
Get-ADUser -Filter *
```

This command retrieves all user accounts from Active Directory.

### Parameter Details

| Parameter   | Description                            |
| ----------- | -------------------------------------- |
| `-Filter *` | Returns all users without restrictions |

---

## Retrieve Additional User Properties

```powershell id="u6f5vw"
-Property *
```

This retrieves all available user attributes, including:

* Email addresses
* Departments
* Titles
* Login information
* Account status

---

## Select Specific User Attributes

```powershell id="o4r8zb"
Select-Object SamAccountName, Name, GivenName, Surname, EmailAddress, Department, Title, Enabled
```

Defines which attributes appear in the exported CSV file.

---

## Export Results to CSV

# 🧩 Chapter 3 — Advanced Export Customization

---

# Export Additional User Attributes

To include additional AD attributes such as:

* Manager
* Last Logon Date

use the following command:

```powershell id="z8p4ne"
Get-ADUser -Filter * -Property * |
Select-Object SamAccountName, Name, GivenName, Surname, EmailAddress, Department, Title, Enabled, Manager, LastLogonDate |
Export-Csv -Path "C:\AD_Users.csv" -NoTypeInformation
```

---

# Export Users From a Specific Organizational Unit (OU)

To target a specific OU, use the `-SearchBase` parameter.

```powershell id="t6w9qb"
Get-ADUser -Filter * -Property * -SearchBase "OU=Employees,DC=example,DC=com" |
Select-Object SamAccountName, Name, GivenName, Surname, EmailAddress |
Export-Csv -Path "C:\AD_Users.csv" -NoTypeInformation
```

---

## Replace the Distinguished Name (DN)

Replace:

```text id="m9c5xy"
OU=Employees,DC=example,DC=com
```

with your actual Organizational Unit path.

---

# Export Only Active Users

To export only enabled accounts:

```powershell id="y4v7ak"
Get-ADUser -Filter {Enabled -eq $true} -Property * |
Select-Object SamAccountName, Name, GivenName, Surname, EmailAddress, Department |
Export-Csv -Path "C:\Active_Users.csv" -NoTypeInformation
```

# 🧪 Chapter 4 — Common Issues & Troubleshooting

---

# Issue: Active Directory Module Not Found

## Cause

The RSAT tools or ActiveDirectory module are not installed.

---

## Resolution

Install:

* RSAT (Remote Server Administration Tools)

Installation methods include:

* Windows Features
* Windows Optional Features
* Microsoft Download Center

---

# Issue: Insufficient Permissions

## Cause

The current account lacks permission to query Active Directory.

---

## Resolution

Ensure the account has:

* Active Directory read permissions
* Proper administrative privileges

---

# Issue: Empty CSV Export

## Possible Causes

| Cause                 | Description                   |
| --------------------- | ----------------------------- |
| Incorrect OU path     | Invalid `SearchBase` value    |
| Filter issue          | Improper filter syntax        |
| Lack of permissions   | Access restrictions           |
| AD connectivity issue | Domain communication problems |

# 🏁 Chapter 5 — Master Final PowerShell Script

---

# Optimized AD Export Script

```powershell id="r2n8fv"
Get-ADUser -Filter * -Property * |
Select-Object SamAccountName,
              Name,
              GivenName,
              Surname,
              EmailAddress,
              Department,
              Title,
              Enabled,
@{
    Name = "OUPath"
    Expression = {

        # Split DistinguishedName into sections
        $ouParts = ($_.DistinguishedName -split ",")

        # Extract OU components only
        $ouComponents = $ouParts | Where-Object { $_ -like "OU=*" }

        # Build top-to-bottom OU structure
        $ouPath = $ouComponents -join " > "

        # Add username at beginning
        ($_.SamAccountName) + " > " + $ouPath
    }
} |
Export-Csv -Path "C:\AD_Users.csv" -NoTypeInformation
```
